# Wypter
![Wypter Logo](https://i.ibb.co/hXsnd4v/Wypter.jpg)
>Wypter is the webapp for controlling your budget.

Based on:
1. Django
2. Django REST framework
3. Chart.js
4. Azure/GCP
5. Bootstrap
6. Redis as a brocker and Celery for scheduled tasks
